# Classroom 2.0 - Interactive Learning Hub

Classroom 2.0 is a modern Android application designed to enhance classroom interaction between students and teachers. Built with **Jetpack Compose** and following a modern dark aesthetic, it provides essential tools for the digital age.

## \ud83d\ude80 Features

### \u2705 Must-Have Functionalities
1. **Digital Attendance (QR Code)**:
   - Teachers can generate dynamic QR codes.
   - Students scan codes to mark their presence instantly.
2. **Live Quiz**:
   - Interactive quiz system for quick "blic" questions during class.
   - Real-time question tracking and option selection.

### \u2728 Personal Touch (Additional Features)
- **Random Student Picker**: Fairly select students for participation with a fun UI.
- **AI Assistant**: Integration concept for explaining complex OOP and programming concepts.
- **Leaderboard**: Gamification system showing top students based on participation and quiz results.
- **Homework Tracker**: Digital list of upcoming assignments and lab reports.

## \ud83c\udfa8 Design
The app features a **Modern Dark Aesthetic**:
- **Deep Navy & Purple Gradient**: Clean and professional look.
- **Surface Cards**: High-contrast elements for better readability.
- **Interactive Grids**: Intuitive navigation from the Dashboard.

## \ud83d\udee0\ufe0f Technical Stack
- **Language**: Kotlin
- **Framework**: Jetpack Compose
- **Navigation**: Compose Navigation
- **Architecture**: Single Activity, multi-screen Compose architecture.
- **Dependencies**: 
  - Material 3
  - Material Icons Extended
  - ZXing (QR Support)
  - Firebase (Concept ready)

## \ud83d\udcc2 Project Structure
- `ui/theme`: Custom colors, typography, and theme definitions.
- `ui/screens`: Implementation of all feature screens.
- `ui/navigation`: NavGraph and screen routing logic.
